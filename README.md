# Movie-Rating-Pipeline
Built an end-to-end PySpark Data Engineering project that processes large-scale movie ratings data for a streaming platform. The pipeline performs data ingestion, cleaning, validation, transformation, aggregation, and analytics using PySpark and Spark SQL.


📌 Project Overview

This project demonstrates how to build a complete data processing pipeline using PySpark.

The pipeline performs:

Data ingestion
Data cleaning
Data transformation
Aggregations
Trend analysis
Reporting dataset generation

The goal is to simulate a real-world streaming company analytics workflow.

🚀 Technologies Used
Python
PySpark
Spark SQL
DataFrames API
CSV File Processing
Databricks

# Analytics Performed

⭐ Average Movie Ratings

Calculate average ratings for each movie.

🔥 Trending Movies

Identify movies with highest engagement.

👥 User Behavior Analysis

Analyze rating activity by users.

🎯 Top Rated Movies

Find highest-rated movies based on average score.
